# mithril-redux-bootstrap

## About

Mithril.js + Redux boilerplate with Bootstrap4.

- Mithril.js
- Redux
- Bootstrap4
- ES6
- JSX
- HotLoader both JavaScript/Bootstrap
- webpack3
- eslint
- dev/production env
- redux-logger
- jest

## Install

```
npm install
```

## Scripts

**Install**

```
npm install
```

**Develop with JavaScript/Bootstrap HotLoader**
```
npm start
```

**Build**
```
npm run build
```

**Build only app**
```
npm run build:app
```

**Build only Boostrap4**
```
npm run build:bootstrap
```
